Algumas palavras sobre mim

Gosto de programar, escrever código mesmo, mas, com objetivo, ou seja, para resolver algum problema. Facilitar a vida de alguma forma. Meu primeiro contato com a programação foi no ano de 2017, através das aulas do digníssimo professor Gustavo Guanabara. Era um curso de Python. Desde então sigo nessa code life. Em 2021, obtive formação tecnóloga em análise e desenvolvimento de sistemas e então me aprofundei nos estudos na área de desenvolvimento web, utilizando Python e o framework Django. Atualmente estou focando meus estudos em Python, na melhoria de técnicas utilizadas no desenvolvimento web com Django e na construção de API's com o framework Django Rest Framework e FastAPI; Gerenciamento de containers com Docker; Machine Learning;
